Unit Overview

This unit introduced the process of software development from planning and design to implementation and testing. I learned how developers create applications that solve real-world problems using structured programming techniques and logical design.

Projects Completed

Calculator Application

I created a calculator program capable of performing arithmetic operations such as addition, subtraction, multiplication, and division. This helped me understand how logic is implemented in software.

Currency Converter

I developed a currency converter that calculates values between different currencies using predefined exchange rates. This improved my understanding of variables and mathematical operations in programming.

Programming Concepts

Variables And Data Storage

I learned how variables are used to store, update, and manage data within a program. This is essential for handling user input and performing calculations.

Selection (If Statements)

I used selection statements to allow programs to make decisions based on conditions, enabling different outputs depending on user input.

Iteration (Loops)

I learned how loops repeat actions efficiently, reducing the need for repeated code and improving program structure and performance.

Functions And Methods

I used functions to create reusable blocks of code, improving organisation and reducing repetition within programs.

Challenges And Problem Solving

One of the most difficult parts of this unit was understanding how different parts of a program work together, especially when debugging errors. I improved by testing code step-by-step, analysing errors carefully, and breaking problems into smaller parts to solve them more effectively.

Skills Development

This unit improved my logical thinking, problem-solving ability, and understanding of how software is structured. I became more confident reading and writing code and understanding how programming concepts work together in real applications.
