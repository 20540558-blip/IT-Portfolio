UNIT 2 — TECHNOLOGY SYSTEMS

Understanding Technology Systems

This unit focused on how hardware and software work together to process data and run computer systems in modern devices.

View PowerPoint

Unit Overview

This unit explored how computer systems are built and how hardware and software work together to process, store, and communicate information.

Hardware Components

CPU

I learned that the CPU processes instructions and acts as the brain of the computer.

RAM

RAM temporarily stores data while programs are running to improve speed and performance.

Storage Devices

I explored HDDs and SSDs and learned how files and operating systems are stored.

Input & Output Devices

I learned how devices such as keyboards, mice, monitors, and printers work together.

Software Systems

System Software

I learned how operating systems manage hardware, memory, and system resources.

Application Software

I explored applications such as browsers, word processors, and media software.

How Systems Process Data

I learned about the Input-Process-Output model and how systems receive, process, store, and display information.

Challenges

Some topics were difficult at first, especially understanding how hardware communicates together, but research and practice improved my confidence.

This unit improved my understanding of how computer systems function and helped me become more confident explaining hardware and software.
