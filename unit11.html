Unit Overview

This unit explored how computer networks are designed and configured to allow communication between devices. I learned how networking technologies support businesses, schools, and organisations by enabling resource sharing, internet access, and secure communication.

Network Design

Star Topology

I designed a star topology network where all devices connect to a central switch. This structure improves reliability because if one connection fails, the rest of the network continues operating normally.

Network Planning

I planned the layout of devices carefully to ensure efficient communication between computers, printers, and networking hardware. This improved my understanding of professional network organisation and design.

Devices And Hardware

Routers

I learned that routers are responsible for connecting different networks together and directing data packets to their destinations efficiently across the internet and local networks.

Switches

Switches allow multiple devices to communicate within the same network by transferring data only to the intended destination device, improving network efficiency.

End Devices

I connected PCs and printers to the network and understood how shared resources allow users to access files, printers, and internet services from multiple devices.

Configuration And Testing

I configured IP addresses, connected networking devices, and tested communication between systems. This required precision because incorrect settings prevented devices from communicating successfully. I also learned how to troubleshoot network issues using logical testing methods.

Challenges And Problem Solving

This was one of the most technically challenging units because even small configuration mistakes could cause the entire network to fail. I improved my problem-solving skills by checking cable connections, reviewing IP settings, and testing communication step-by-step until issues were resolved.

Skills Development

This unit improved my technical understanding of networking systems and strengthened my ability to configure and troubleshoot networks independently. I also improved my patience, analytical thinking, and attention to detail while solving networking problems.

Reflection

Overall, this unit gave me a much stronger understanding of how real-world computer networks operate. Although challenging, it improved my confidence configuring devices, solving technical issues, and understanding how organisations rely on networks for communication and data sharing.
