Unit Overview

This unit covered the fundamentals of website development, including structuring content with HTML and styling pages using CSS. I developed an understanding of how websites are built and how design choices affect usability and user experience.

Technologies Used

HTML (Structure)

I used HTML to structure content on web pages, including headings, paragraphs, images, and navigation menus to ensure clear organisation.

CSS (Design & Styling)

I used CSS to control the appearance of the website, including layout, colours, spacing, and interactive effects such as hover animations.

Key Features Implemented

Navigation System

I created a consistent navigation menu across all pages, allowing users to move easily between different sections of the portfolio.

Responsive Layout

I designed a flexible layout that adapts to different screen sizes, improving accessibility on both desktop and mobile devices.

Visual Design

I applied a consistent colour scheme, spacing, and typography to create a professional and visually appealing website design.

User Experience (UX)

I focused on making the website easy to read, navigate, and visually clear. I improved UX by using consistent layouts, spacing content properly, and ensuring that all interactive elements are easy to identify and use.

Challenges And Problem Solving

One of the main challenges was understanding how HTML and CSS interact to control layout and styling. I also had to debug issues where small coding errors affected the overall page design. I solved these by testing changes step-by-step and reviewing code structure carefully.

Skills Development

This unit improved my front-end development skills, creativity, and understanding of web design principles. I also developed stronger problem-solving abilities and became more confident in building structured, user-friendly websites.
